Инструкция по использованию скрипта Triggerbot:


Скачайте и установите AutoHotkey отсюда: https://www.autohotkey.com/

Скачайте скрипт Triggerbot и сохраните его с расширением .ahk.

Запустите скрипт правым кликом мыши по файлу и выбрав "Запустить скрипт".

Чтобы активировать триггер, нажмите клавишу "Up", а чтобы выключить - нажмите клавишу "Down".

По умолчанию клавиша выстрела - "P", но ее можно изменить, отредактировав значение переменной "shoot_key" в скрипте.

Чтобы настроить чувствительность триггера, измените значение переменной "pixel_sens" в скрипте. Чем выше значение, тем чувствительнее триггер.

Чтобы изменить цвет пикселя цели, измените значение переменной "pixel_color" в скрипте. Цвет указывается в шестнадцатеричном формате (например, 0xFF0000 для красного цвета).

Чтобы изменить задержку между выстрелами, измените значение переменной "tap_time" в скрипте. Задержка указывается в миллисекундах.

Чтобы изменить размер области поиска пикселя цели, измените значение переменной "pixel_box" в скрипте.

Чтобы выйти из скрипта, нажмите клавишу "Delete".
