Инструкция по использованию скрипта Triggerbot:


Убедитесь, что у вас установлен AutoHotkey (AHK). Если нет, загрузите и установите его с официального сайта: https://www.autohotkey.com/.

Скачайте файл скрипта (.ahk) и сохраните его на вашем компьютере.

Откройте настройки вашей игры и измените цвет обводки прицела на желтый. Это необходимо для того, чтобы скрипт мог правильно обнаруживать пиксели цели.

Откройте файл скрипта в текстовом редакторе, чтобы внести необходимые изменения в настройки:

boxSize: размер области поиска вокруг курсора (по умолчанию - 3).

sensitivity: чувствительность обнаружения цвета (по умолчанию - 50).

targetColor: цвет, который нужно обнаружить (по умолчанию - 0xFEFE40, желтый).

clickDelay: задержка между щелчками в миллисекундах (по умолчанию - 150).

holdKey: клавиша, которую нужно удерживать для работы скрипта в режиме HoldMode (по умолчанию - "SPACE").

clickModeHotkey, holdModeHotkey, turnOffHotkey, terminateHotkey: горячие клавиши для переключения режимов, выключения и завершения работы скрипта (по умолчанию - "Left", "Right", "Up", "Down").

Сохраните изменения в файле скрипта.

Запустите скрипт, двойной щелкнув по файлу (.ahk).

Во время игры используйте горячие клавиши для переключения режимов, выключения и завершения работы скрипта:
Левая стрелочка (←) - переключает режим постоянного щелчка (ClickMode).
Правая стрелочка (→) - переключает режим удержания (HoldMode).
Верхняя стрелочка (↑) - выключает скрипт.
Нижняя стрелочка (↓) - завершает работу скрипта.

При необходимости отредактируйте файл скрипта, чтобы изменить настройки или горячие клавиши.
